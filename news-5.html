CVE-2026-6598 | langflow-ai langflow fino a 1.8.3 Project Creation Endpoint projects.py create_project/encrypt_auth_settings rivelazione di informazioni

È stata rilevata una vulnerabilità classificata come problematico in langflow-ai langflow fino a 1.8.3. Risulta coinvolta una funzione sconosciuta del file src/backend/base/Langflow/api/v1/projects.py del componente Project Creation Endpoint. Attraverso la manipolazione del parametro auth_settings di un input sconosciuto per mezzo di una vulerabilità di classe rivelazione di informazioni. Questo punto di criticità è identificato come CVE-2026-6598. Dalla rete può partire l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358233
