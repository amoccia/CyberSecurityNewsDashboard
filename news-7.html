CVE-2026-6596 | langflow-ai langflow fino a 1.1.0 API Endpoint endpoints.py create_upload_file escalationi di privilegi

Un punto di debole di livello critico è stato rilevato in langflow-ai langflow fino a 1.1.0. É interessato la funzione create_upload_file del file src/backend/base/Langflow/api/v1/endpoints.py del componente API Endpoint. La gestione determina escalationi di privilegi. Questa vulnerabilità viene indicata come CVE-2026-6596. Nella rete si effettua l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358231
