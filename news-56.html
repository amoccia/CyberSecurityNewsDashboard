CVE-2026-40485 | ChurchCRM fino a 7.1.x Public API Login Endpoint /api/public/user/login rivelazione di informazioni (GHSA-x2qh-xmhq-4jpx / EUVD-2026-23599)

In ChurchCRM fino a 7.1.x stata rilevata una vulnerabilità di livello problematico. Da questa vulnerabilità è interessato una funzione sconosciuta del file /api/public/user/login del componente Public API Login Endpoint. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe rivelazione di informazioni. Questa vulnerabilità viene indicata come CVE-2026-40485. Nella rete si effettua l'attacco. Nessun exploit disponibile. Si suggerisce di aggiornare il componente vulnerabile.
