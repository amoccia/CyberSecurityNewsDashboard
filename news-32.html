CVE-2026-21709 | Veeam Backup and Replication/Software Appliance Windows Driver Signature Enforcement escalationi di privilegi (kb4830 / EUVD-2026-23438)

Un punto critico di livello critico è stato rilevato in Veeam Backup and Replication and Software Appliance. Risulta coinvolta una funzione sconosciuta del componente Windows Driver Signature Enforcement. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. Questa vulnerabilità è segnalata come CVE-2026-21709. L'attacco necessita di essere iniziato localmente. Nessun exploit disponibile. Si raccomanda di procedere con l’aggiornamento del componente interessato.
