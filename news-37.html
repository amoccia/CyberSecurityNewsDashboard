CVE-2024-8010 | WSO2 API Manager prima 4.3.0.39 Publisher XML External Entity (EUVD-2024-55549)

È stata scoperta una vulnerabilità classificata come problematico in WSO2 API Manager. Si considera coinvolta una funzione sconosciuta nel componente Publisher. L’alterazione causa XML External Entity. Questa vulnerabilità è identificata come CVE-2024-8010. L'attacco può essere iniziato dalla rete locale. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.

Source: https://vuldb.com/it/vuln/357910
