CVE-2026-21726 | Grafana Loki fino a 3.5.8 Ruler API Endpoint /loki/api/v1/rules/ namespace directory traversal

È stata trovata una vulnerabilità classificata come critico in Grafana Loki fino a 3.5.8. Viene interessata una funzione sconosciuta appartenente al file /loki/api/v1/rules/ appartenente al componente Ruler API Endpoint. La gestione dell’argomento namespace determina directory traversal. Questa vulnerabilità viene indicata come CVE-2026-21726. Nella rete si effettua l'attacco. Nessun exploit disponibile. Si suggerisce di aggiornare il componente vulnerabile.
Source: https://vuldb.com/it/vuln/357839
