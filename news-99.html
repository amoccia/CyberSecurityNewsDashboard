CVE-2026-34393 | weblate fino a 5.16 API Endpoint escalationi di privilegi (GHSA-3382-gw9x-477v)

Un punto di criticita di livello critico è stato rilevato in weblate fino a 5.16. Riguarda una funzione sconosciuta del componente API Endpoint. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe escalationi di privilegi. Questa vulnerabilità è segnalata come CVE-2026-34393. L'attacco si effettua con la rete. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.
